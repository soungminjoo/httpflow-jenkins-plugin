Runs httpflow request to reproduce http request which is executed in web browser.
Request contents should have grammar like below. It can be copied easily from chrome browser (developer tools > Newwork > headers)
GET https://www.google.com/search?q=jenkins
Content-Type: application/x-www-form-urlencoded
:authority: www.google.com
:method: GET
:path: /search?q=jenkins
:scheme: https
Content-Type: application/x-www-form-urlencoded
